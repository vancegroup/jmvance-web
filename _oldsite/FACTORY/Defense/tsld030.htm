Time Manipulation of the Simulation

  • Process of jumping through time

    • The virtual world is reset
    • All parts are detached
    • Fixtures, doors, lifts, etc. set back to initial positions
    • Time is set back to zero
    • part attributes set back to original state

  • Function jump_to_time (similar to part_process) is called for every part

    • Different in that there is no animation (instantaneous)
